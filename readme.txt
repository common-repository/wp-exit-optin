Wordpress Exit Optin is a marketing tools to help you get your visitor email address when they are about to leave your site.

== Description ==

Wordpress Exit Optin is developed to help you get your website visitors attention when they are about to leave your site. This can be used in any creative way you can think of - give promotion code with discounts, subscribe to newsletter, pool, etc.

This is plugin is very simple and lightweight. You only need to input the Popup Title, Description, and the form you are using to get the email. You can use Contact Form 7 or Gravity Forms, any for that has shortcode can do.

== Installation ==

1. Use the Plugins -> Add New menu in WordPress to install the plugin or upload the wp-exit-optin folder to the /wp-content/plugins/ directory.
1. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= When the popup box shown? =

The popup box will appear with these conditions:

* User is not logged in
* User about to exit your website
* The user never see this popup before this day
